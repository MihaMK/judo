# Judo Club System

Enterprise web application foundation for operating a competitive judo club.

## Foundation Principles

- Feature-first architecture.
- Server-validated workflows.
- Mobile-first operational screens.
- No business logic in UI components.
- Immutable/auditable history for critical future workflows.

## Current Scope

This initialization includes only the project foundation: Next.js App Router, TypeScript, Tailwind, Supabase client structure, role-aware shell, shared UI primitives, loading/error boundaries, and environment configuration.

Attendance, payments, competitions, notifications, athlete history, and database business schema are intentionally not implemented yet.
